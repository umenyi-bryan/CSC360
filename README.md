# CSC360 - Computer Science Student Portal

A comprehensive student portal for the Computer Science department featuring assignment management, code editor, and admin dashboard.

## Features
- Student dashboard with assignment tracking
- VS Code-like integrated code editor (Monaco Editor)
- Admin dashboard for H.O.D (Dr. Comfort Amalahu Ed)
- Assignment submission system
- Real-time code preview
- Responsive design for all devices

## Technologies Used
- HTML5, CSS3, JavaScript
- Monaco Editor (VS Code engine)
- Font Awesome icons
- Deployed on Vercel

## Setup
1. Clone the repository
2. Open `index.html` in a browser
3. For admin view, click "Admin" in the top right corner
